== Description ==

Custom Login Suite, you can be thoroughly customize the behavior of around login of WordPress.

For example, you can login at the another URL that is not "wp-login.php", you can add your own items to the user registration, you can incorporate the CAPTCHA authentication during password reset, and of course you can easy change the design of login screen.

[Please visit here for more description of the plugin](http://ka2.org/).

== Installation ==

1. Upload `custom-login-suite` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to the Custom Login Suite submenu in setting menu

That's it. Now you can easily start setting any options.
